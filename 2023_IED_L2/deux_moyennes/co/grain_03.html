Test de normalité

Le test de Shapiro-Wilk est un test permettant de savoir si une série de donées suit une loi normale. Essayez de tester la normalité d’un échantillon via Jamovi.

  • Ouvrir les données dans Jamovi[1]

  • Choisissez le menu « Paired Samples T-Test »[2]

  • Choisissez le test de normalité « Normality test »[3]
