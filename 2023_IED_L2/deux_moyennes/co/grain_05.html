Analyser le résultat de test de Shapiro-Wilk

Le test de Shapiro-Wilk est un test permettant de savoir si une série de données suit une loi normale.

HO : la normalité est respectée

H1 : la normalité n'est pas respectée

Ici, p > 0,05, la p-valeur n'est pas significative. L'échantillon suit donc une loi normale.

Nous allons donc utiliser le test de t de student.
